# Data Preporcessing
This project focuses on data preprocessing and cleaning techniques applied to a heart disease dataset, followed by building a basic machine learning model for prediction. The goal is to transform raw, inconsistent data into a clean and structured format suitable for analysis and modeling.

# Dataset
Data set is sourced from - https://www.kaggle.com/datasets/abhishekkaggle8585/heart-patients-dirty-dataset

# Features
* Handling missing values (numerical & categorical)
* Removing duplicate records
* Fixing inconsistent data formats
* Outlier detection and treatment using IQR
* Data encoding (Label Encoding & One-Hot Encoding)
* Feature scaling using StandardScaler
* Model training using Logistic Regression
* Feature importance analysis

# Results
* Achieved ~61.9% accuracy using Logistic Regression
* Balanced performance with minimal overfitting
* Identified key contributing features using feature importance
# Model Performance
* Accuracy: 61.9%
* F1 Score: 0.074
* ROC-AUC Score: 0.58

NOTE - The accuracy of the model is low because the dataset has very limited samples and it is created only for the purpose to perform the basic preprocessing techniques.
